# 가격 비교 검색 서비스

한국 주요 쇼핑몰(네이버쇼핑, 쿠팡, 11번가, G마켓, 옥션, 인터파크, SSG 등)에서 상품의 최저가를 비교하는 웹 애플리케이션입니다.

## 🚀 주요 기능

- **다중 쇼핑몰 검색**: 주요 쇼핑몰을 한 번에 검색
- **최저실현가 계산**: 기본가 + 옵션가 + 배송비 - 할인금액
- **가격 비교**: 최저가부터 정렬된 비교 테이블
- **분석 요약**: 평균 가격, 무료배송 비중, 할인율 통계
- **반응형 디자인**: 모바일, 태블릿, 데스크톱 지원

## 📊 제공 정보

각 상품별로 다음 정보를 제공합니다:
- 쇼핑몰명
- 상품명
- 옵션 (색상, 용량, 사이즈 등)
- 정가 (기본가)
- 할인가 (쿠폰/즉시할인 반영)
- 배송비
- **최저실현가** (할인가 + 배송비)
- 상품 링크

## 🎯 사용 방법

1. 검색창에 찾고자 하는 상품명 입력
2. "검색하기" 버튼 클릭
3. 가격 비교 결과 확인
4. "바로가기" 버튼으로 해당 쇼핑몰 방문

## 📝 참고사항

현재 버전은 데모 데이터를 사용합니다. 실제 쇼핑몰 데이터를 가져오려면 백엔드 API 서버가 필요합니다.

### 실제 구현을 위한 추가 작업

   - 웹 스크래핑 또는 쇼핑몰 API 연동

2. **데이터베이스**
   - 검색 기록 저장
   - 가격 변동 추적

3. **고급 기능**
   - 가격 변동 알림

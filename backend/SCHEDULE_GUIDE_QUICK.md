# ⏰ 자동 스케줄링 - 빠른 가이드

## 🚀 1시간마다 자동 실행 설정

### 단 한 번만 실행하면 끝!

```cmd
cd C:\Users\netwo\Documents\find-item2\backend
schedule_hourly.bat
```

**⚠️ 중요:** 우클릭 → "관리자 권한으로 실행"

---

## ✅ 설정 완료 후

### 자동으로 실행됨:
- 09:00, 10:00, 11:00, ... 23:00
- **매 시간마다** 가격 수집
- **하루 15회** 자동 업데이트

### 사용자는:
- 아무것도 안 해도 됨
- 웹만 열면 항상 최신 가격
- 완전 자동화!

---

## 🎛️ 관리 명령어

### 상태 확인
```cmd
schedule_status.bat
```

### 중지
```cmd
schedule_stop.bat
```
(관리자 권한 필요)

### 간격 변경
```cmd
schedule_custom.bat
```
30분 / 1시간 / 2시간 / 3시간 / 6시간 / 12시간 / 매일 선택

## 🔍 작동 확인

### Windows 작업 스케줄러
```
Win + R → taskschd.msc
→ "PriceScraperHourly" 찾기
```

### 마지막 실행 확인
```cmd
schedule_status.bat
```

---

## ⚠️ 문제 해결

### "관리자 권한 필요" 오류
→ 배치 파일 우클릭 → "관리자 권한으로 실행"

### 작업이 실행 안 됨
→ `schedule_status.bat`로 상태 확인
→ 작업 스케줄러에서 수동 테스트

### 배치 실행은 되는데 실패함
→ `run_scraper.bat` 수동 실행 테스트
→ 인터넷 연결 확인
→ Chrome 업데이트
